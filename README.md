# E-Commerce Data Analysis Project

## Project Overview

This project presents an end-to-end E-Commerce Sales Analysis solution developed using SQL Server, Power BI, Power Query, DAX, and Excel. The objective of this project is to analyze sales performance, profitability, customer segments, and regional trends to derive actionable business insights.

---

## Tools & Technologies Used

* SQL Server Management Studio (SSMS)
* SQL
* Power BI
* Power Query
* DAX
* Microsoft Excel

---

## Dataset

* Sample Superstore Dataset

---

## Project Workflow

1. Data Collection and Import
2. SQL Data Analysis
3. Data Cleaning using Power Query
4. Data Modeling in Power BI
5. DAX Measure Creation
6. Dashboard Development
7. Business Insights Generation

---

## SQL Analysis Performed

* Total Sales Analysis
* Total Profit Analysis
* Sales and Profit by Region
* Sales and Profit by Category
* Top and Bottom Performing States
* Customer Segment Analysis
* Discount Impact on Profitability
* Profit Status Classification using CASE WHEN

---

## Power BI Dashboard Pages

### Page 1: Executive Dashboard

Features:

* KPI Cards (Sales, Profit, Quantity, Profit Margin)
* Sales by Region
* Profit by Region
* Sales by Category
* Profit by Category
* Top Profitable States
* Discount vs Profit Analysis
* Interactive Slicers

### Page 2: Category & Segment Analysis

Features:

* Sales by Sub-Category
* Profit by Sub-Category
* Sales by Segment
* Profit by Segment
* Interactive Category and Segment Slicers

---

## Key Business Insights

* West region generated the highest sales and profit.
* Technology category delivered the highest profitability.
* Consumer segment contributed the maximum revenue.
* California and New York were the most profitable states.
* Higher discounts negatively impacted profitability.
* Furniture category showed comparatively lower profitability.
